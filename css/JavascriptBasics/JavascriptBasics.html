JavaScript Basics



Javascript is a cross-platform object oriented scripting language. Javascipt is extremely popular for a variety of reasons. It is a small and lightweight language allowing maximum flexibility for developers to take it in a bunch of different directions. Javascript loves inside a host environment (a web browser or Node server), it can be connected to the objects of these environments to provide programmatic control over them.

  • Variable Declaration Javascript variables are containers for storing data values - imagine a cup you fill with coffee, the cup holds the coffee, a variable holds a alue. All JavaScript variables must be identified with unique names. These unique names are called identifiers.var x;

  • Variable Assignment Assignment operators assign values to JavaScript variables - our cup can now have coffee poured in it, giving our variable a value to hold. The = assignment operator assigns a value to a variable. var x =10;
  • Data Types

    Data types are an important concept; to be able to operate on variables you need to know the data type. There are six data types that are JavaScript primitives: Boolean - true or false; null - null aka nothing; Number - 42 or 3.13159; String - "Coding Dojo Rocks"; Array - [1, 'Coding', 1, 'Dojo'] and Object - {first_name: 'Jane, last_name:'Doe'}
